### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、销售员、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 雪具商品
5. 公告信息
6. 客服中心
7. 购物车
8. 个人中心
9. 我的订单
10. 我的地址
11. 我的收藏



**销售功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 雪具分类管理
5. 雪具商品管理
6. 进货记录管理
7. 退货记录管理
8. 订单管理




**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 销售员管理
4. 用户管理
5. 雪具分类管理
6. 雪具商品管理
7. 进货记录管理
8. 退货记录管理
9. 系统管理（公告信息、客服中心、系统简介、轮播图管理）
10. 订单管理
